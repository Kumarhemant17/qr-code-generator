# 🔳 QR Code Generator

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-yellow)

A simple and responsive QR Code Generator built using **HTML, CSS, and JavaScript**.  
It allows users to generate QR codes from text or URLs instantly with additional features like download and size control.

---

## 🚀 Live Demo

👉 https://kumarhemant17.github.io/qr-code-generator/

---

## ✨ Features

- 🔹 Generate QR codes from text or URLs
- 🔹 Adjustable QR size using slider
- 🔹 Download QR code as PNG
- 🔹 Copy input text to clipboard
- 🔹 Loading indicator for better UX
- 🔹 Error animation for empty input
- 🔹 Clean glassmorphism UI with gradient background

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Gradient, Glassmorphism)
- JavaScript (DOM Manipulation, Events)
- QR Code API
